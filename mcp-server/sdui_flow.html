✨ The SDUI Magic Flow ✨

Watch how a natural language query transforms into a Native Android UI in milliseconds!

How the Magic Happens:
When the user types a query (e.g. "Test" or "Show Accounts"), the Android app sends it to ai_screen.php.
The PHP script quickly checks MySQL. If it's a new query, it gathers the user's data and sends it to Gemini AI along with our strict widget rules.
Gemini reads the rules, acts as a UI designer, and builds a strict JSON tree using `lazy_column`, `balance_card`, etc.
PHP receives the JSON, caches it in MySQL for instant future loads, and sends it to Android where RendererFactory paints it natively!
